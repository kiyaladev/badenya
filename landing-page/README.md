# Badenya Landing Page

Site web vitrine pour l'application Badenya.

## Stack Technique

- **Framework**: React 19
- **Build Tool**: Vite
- **Language**: TypeScript
- **UI**: Tailwind CSS (à installer)
- **Animations**: Framer Motion (à installer)

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application démarre sur http://localhost:5173

## Build Production

```bash
npm run build
npm run preview
```

- Features principales
- Comment ça marche
